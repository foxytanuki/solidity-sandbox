# Solidity Sandbox

A research repository for exploring Solidity development ecosystem.

## Topics

- **Standards** — ERC specifications (ERC-20, ERC-721, ERC-1155, etc.)
- **Developer Tools** — Hardhat, Foundry, and related tooling
- **Protocol Demos** — Example integrations with DeFi protocols

## Structure

```

scripts/        # Automation and deployment scripts
test/           # Test suites
```
